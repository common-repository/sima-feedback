== Frequently Asked Questions ==
What does Sima mean?
Sima is a Persian word that means face (or beautiful face).

What does Sima do?
Sima is a visual feedback tool that enables users to provide clear and actionable feedback on websites. Sima also integrates with product management tools to help manage and track feedback.

== Third-Party Services == 

**Sima Feedback** integrates with [Sima Feedback](https://www.simafeedback.com) to provide visual feedback features on your website. When activated, the plugin loads an external script from `https://www.simafeedback.com/sima.js`.
